# Batelu Website
**Batelu** is a community-driven language created on May 3, 2025.
You can contribute to this language right now by going to Home, joining the discord server, familiarizing yourself with the language's mechanics/rules, and then creating polls for other fellow conlangers to vote on to change the language!

When a word is added or changed, go to [this link](https://github.com/Sushii64/batelu/blob/main/src/pages/words.json).
Likewise, for grammar rules, go to [this link](https://github.com/Sushii64/batelu/blob/main/src/pages/items.json).

(for the date, use the date of the poll <ins>ending</ins>.)
